# Multi-Platform Bot

Бот для публикации постов в несколько Telegram каналов.

## Установка

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать: `venv\Scripts\activate` (Windows) или `source venv/bin/activate` (Linux/Mac)
4. Установить зависимости: `pip install -r requirements.txt`
5. Скопировать `.env.example` в `.env` и заполнить данные
6. Запустить: `python main.py`

## Команды

- `/start` - Начать работу
- `/add_community` - Добавить канал
- `/my_communities` - Мои каналы
- `/new_post` - Создать пост
- `/my_posts` - Мои посты
- `/admin` - Админ панель (только для админов)

## Функции

✅ Публикация в несколько Telegram каналов
✅ Сохранение кастомных эмодзи
✅ Выбор каналов для публикации
✅ История постов
✅ Админ панель со статистикой
